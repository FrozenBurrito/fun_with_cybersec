## Virtual Environment Setup
* Linux
```
git clone https://github.com/FrozenBurrito/fun_with_cybersec.git
cd fun_with_cybersec
python -m venv .venv
sudo -s
source .venv/bin/activate
pip install -r requirements.txt
```
* Windows
```
git clone https://github.com/FrozenBurrito/fun_with_cybersec.git
cd fun_with_cybersec
python -m venv .
scripts\activate.bat 
pip install -r requirements.txt
```
